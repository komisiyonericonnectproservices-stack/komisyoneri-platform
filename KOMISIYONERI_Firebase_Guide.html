Uko Files Zigomba Kugaragara mu Mubyimba
Mbere yo gutangira, tegura aya mafiles mu mubyimba (folder) komisiyoneri/ kuri mudasobwa yawe:
📁 komisiyoneri/ (principal folder yawe)
📁 public/ (Firebase igashyira files zose hano)
📄 index.html ← KOMISIYONERI_Platform.html
📄 manifest.json ← PWA manifest
📄 robots.txt ← Google crawler
📄 sitemap.xml ← Google Search
📄 404.html ← Page ntiboneka (optional)
📄 firebase.json ← Firebase config (igakorwa vuba)
📄 .firebaserc ← Project ID (igakorwa vuba)
Intambwe: Rename KOMISIYONERI_Platform.html ugore index.html — hanyuma ushyire mu folder public/
1
Jya kuri Google Firebase Console — Fungura Konti
Jya kuri firebase.google.com ukoresha konti yawe ya Google (Gmail). Firebase igabanuka Google Project.
  • Kanda "Go to Console" hanyuma "Create a Project"
  • Ita project yawe: komisiyoneri cyangwa komisiyoneri-rw
  • Google Analytics: Enable (ni ingenzi kuri SEO)
  • Kanda "Create Project" — birakozwe mu masegonda 30
⚠️ Ijambo ry'Ahabanza: Project ID (nk. komisiyoneri-rw) ntishobora guhinduka nyuma yo kuyiremaho. Hitamo neza!
2
Shyira Node.js na Firebase CLI kuri Mudasobwa Yawe
Firebase CLI (Command Line Interface) niyo ikora deploy. Igomba Node.js kugira ngo ikore.
Windows — Kanda kandi urokote:
# 1. Shyira Node.js (koresha installer):
https://nodejs.org/en/download   # ← Fyata iyi link mu browser

# 2. Emeza ko Node yashyizwe (fungura Command Prompt / Terminal):
node --version     # Igomba kugaragara: v18.x.x cyangwa hejuru
npm --version      # Igomba kugaragara: 9.x.x cyangwa hejuru

# 3. Shyira Firebase CLI:
npm install -g firebase-tools

# 4. Emeza ko Firebase yashyizwe:
firebase --version  # Igomba kugaragara: 13.x.x
💡 Niba ukoresha Mac, binyuranye gusa aha: sudo npm install -g firebase-tools
3
Login Firebase na Google Account Yawe
Injira Firebase CLI ukoresha Gmail yawe ya Google (imwe wishyiriyeho Firebase Console).
# Fungura Terminal/Command Prompt, wandike:
firebase login

# Browser izafunguka — injira na Gmail yawe
# Nyuma yo kwinjira, uzabona:
✔  Success! Logged in as you@gmail.com
4
Tegura Firebase mu Mubyimba wa KOMISIYONERI
Jya mu mubyimba komisiyoneri/ (aho ufite files zose), hanyuma utangize Firebase.
# Jya mu mubyimba wawe (hindura inzira nk'uko iri kuri mudasobwa yawe):
cd C:\Users\YourName\Documents\komisiyoneri   # Windows
cd ~/Documents/komisiyoneri                    # Mac/Linux

# Tangiza Firebase mu mubyimba:
firebase init hosting

# Ibibazo Firebase izakubaza (subira uku):
? Which Firebase project do you want to use?
  → komisiyoneri-rw  (hitamo project yawe)

? What do you want to use as your public directory?
  → public           (andika "public" hanyuma Enter)

? Configure as a single-page app (rewrite all urls to /index.html)?
  → Yes              (andika "y" hanyuma Enter)

? Set up automatic builds and deploys with GitHub?
  → No               (andika "n" hanyuma Enter)

? File public/index.html already exists. Overwrite?
  → No               (andika "n" — reka index.html yawe)
✅ Nyuma y'ibibazo, Firebase izaremaho firebase.json na .firebaserc mu mubyimba wawe.
5
Hindura firebase.json — Wongere Ibyiza
Fungura firebase.json ukoresha Notepad cyangwa VS Code, usimbuze iki:
{
  "hosting": {
    "public": "public",
    "ignore": ["firebase.json", "**/.*", "**/node_modules/**"],
    "rewrites": [
      { "source": "**", "destination": "/index.html" }
    ],
    "headers": [
      {
        "source": "**/*.html",
        "headers": [
          { "key": "Cache-Control", "value": "public, max-age=3600" },
          { "key": "X-Content-Type-Options", "value": "nosniff" },
          { "key": "X-Frame-Options", "value": "SAMEORIGIN" },
          { "key": "X-XSS-Protection", "value": "1; mode=block" },
          { "key": "Referrer-Policy", "value": "strict-origin-when-cross-origin" }
        ]
      },
      {
        "source": "**/*.@(js|css|svg|png|jpg|jpeg|gif|webp|ico|woff2)",
        "headers": [
          { "key": "Cache-Control", "value": "public, max-age=31536000, immutable" }
        ]
      }
    ],
    "cleanUrls": true,
    "trailingSlash": false
  }
}
💡 Headers ziri hejuru zikangurira security na speed — byose biri mu Google Lighthouse score yawe!

Deploy! — Ohereza Platform kuri Google
Iki ni intambwe y'ingirakamaro — ohereza files zose kuri Firebase.
# Mu mubyimba wa komisiyoneri/ wandike:
firebase deploy --only hosting

7
Shyira Domain Yawe — komisiyoneri.com
Firebase itanga SSL certificate ubuntu kuri domain yawe. Aya ni amabwiriza yo guhuza www.komisiyoneri.com:
  • Jya kuri Firebase Console → Hosting → Add custom domain
  • Andika komisiyoneri.com hanyuma kanda Continue
  • Firebase izakuha DNS records 2 (A records cyangwa TXT record)
  • Jya kuri domain registrar yawe (GoDaddy, Namecheap, Rwanda...) ushyireho records
  • Tegereza amasaha 24-48 — SSL certificate iraboneka Ubuntu
# DNS Records Firebase izakuha (ingero):
Type    Host     Value
A       @        151.101.1.195
A       @        151.101.65.195
CNAME   www      komisiyoneri-rw.web.app
⚠️ DNS records zishobora gufata amasaa 24-48 kugira ngo zikorere. Birashoboka!
8
Iyandikishe kuri Google Search Console
Google Search Console niyo ikemura ko Google ibona sitewayisitiwe yawe neza kandi iyandike mu search results.
  • Jya kuri search.google.com/search-console
  • Kanda "Add property" → Shyira https://www.komisiyoneri.com
  • Emeza ownership (Firebase igakorwa vuba cyangwa HTML tag)
  • Kanda "Sitemaps" → Shyira https://www.komisiyoneri.com/sitemap.xml
  • Tegereza amasaa 24-72 — Google itangira guhuza platform yawe!
✅ Nyuma y'ibyumweru 2-4, uzabona KOMISIYONERI mu results za Google Search kuri "inzu Kigali", "kugura inzu Rwanda", na keywords zindi!
